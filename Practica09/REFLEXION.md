# Reflexion

## 1. Serializacion

La serializacion permite guardar objetos completos en un archivo.

Es util cuando quieres guardar datos complejos, no solo texto.


## 2. BufferedReader

Se usa porque es mas eficiente que leer caracter por caracter.

Mejora el rendimiento al trabajar con archivos grandes.


## 3. Cerrar archivos

Si no se cierran los archivos pueden ocurrir errores o perdida de datos.

En este programa use try-with-resources para cerrar automaticamente los archivos.
