# Bitacora de uso de IA

## Prompt 1
Pregunte como usar clases abstractas en Java para mi programa.

## Respuesta
La IA explico que son clases base que no se pueden instanciar.

## Adaptacion
Use eso para crear la clase Dispositivo.

## Aprendizaje
Aprendi que las clases abstractas sirven para compartir codigo.


## Prompt 2
Pregunte como funcionan las interfaces en java.

## Respuesta
La IA explico que sirven para definir metodos que otras clases deben implementar.

## Adaptacion
Cree las interfaces Conectable, Reproducible y Cargable.

## Aprendizaje
Aprendi a separar funcionalidades en interfaces.


## Prompt 3
Pregunte como combinar interfaces.

## Respuesta
La IA explico que una clase puede implementar varias interfaces.

## Adaptacion
Use diferentes combinaciones en Celular, Bocina y Laptop.

## Aprendizaje
Aprendi que no todas las clases deben implementar lo mismo.
