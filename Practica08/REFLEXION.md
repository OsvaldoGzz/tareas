# Reflexion

## 1. Uso de colecciones

Use diferentes colecciones porque cada una tiene ventajas.

Si usara ArrayList para todo, algunas operaciones serian mas lentas, como busquedas.

## 2. Comparable vs Comparator

Comparable se usa cuando quieres un orden natural en la clase.

Comparator se usa para crear diferentes tipos de orden.


## 3. Streams

Un stream permite trabajar con datos de forma mas simple.

Es mas legible que un for porque usa menos codigo y se entiende mejor.
