# Reflexion

## 1. Diferencia entre override y overload

Override es cuando una clase hija cambia el comportamiento de un metodo del padre.
Por ejemplo, mostrarDatos en cada figura.

Overload es cuando un metodo tiene el mismo nombre pero diferentes parametros.
Por ejemplo, cambiarRadio en Circulo.


## 2. Uso de instanceof

Se usa instanceof para verificar el tipo antes de hacer casting.
Esto evita errores como ClassCastException.

## 3. Clase abstracta

No se puede instanciar directamente.
Sirve como base para otras clases.


## Experiencia

El polimorfismo ayuda a trabajar con diferentes objetos de forma general.
Pero a veces puede ser confuso si no se entiende bien el tipo real del objeto.
