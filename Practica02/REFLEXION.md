# Reflexion

## 1. Diferencia entre clase y objeto

Una clase es un molde y un objeto es una instancia.

En mi caso, Cancion es la clase y las canciones son objetos para el programa.


## 2. Por que use 3 constructores

Para crear objetos de diferentes maneras y poder hacer lo que se pidio.
Uno con valores por defecto, otro con datos basicos y otro completo.

## 3. Si no hubiera constructores

Java crea uno vacio automaticamente, pero solo si no se define ninguno y eso nos facilita un oici oero tiene defectos.
