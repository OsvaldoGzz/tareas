# Practica 2 - Clases y Objetos

## Descripcion
En esta practica se trabajo con clases y objetos en Java.

## Dominio elegido
Cancion / Playlist.

## Justificacion
Se eligio este dominio porque es facil de entender y permite trabajar con atributos, constructores y metodos y algo que me gusta hacer siempre es escuchar musica.

## Estructura
- Cancion.java: clase principal
- Playlist.java: clase contenedora
- Main.java: ejecucion del programa

## Funcionamiento
Se crean 5 canciones y se agregan a una playlist.
Tambien se prueban metodos como mostrar, buscar y modificar datos para el programa.
