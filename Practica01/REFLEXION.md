# Reflexion

## 1. JDK, JRE y JVM

El JDK sirve para programar en Java, ya que incluye herramientas y utilizaremos eso en el semestre.
El JRE sirve para ejecutar los programas que haremos en el semestre.
La JVM es la que interpreta el programa para poder correrlo.

Se instala el JDK porque incluye todo.

## 2. Write once, run anywhere

Java funciona en diferentes sistemas gracias a la JVM.

Esto permite usar el mismo programa en distintas computadoras.


## 3. System.out.println

Sirve para imprimir texto en consola.

System es la clase, out es la salida y println es el metodo.
