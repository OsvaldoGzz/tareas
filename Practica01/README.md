# Practica 1 - Introduccion a Java

## Descripcion
En esta practica se realizo la configuracion del entorno de desarrollo en Java y la creacion de un programa basico .

## Instalacion
Se instalo el JDK y se configuro el IDE NetBeans para poder desarrollar programas en Java y trabajar en el semestre ahi.

## Programa realizado
Se creo un programa HolaMundo que ensena lo siguiente como mensaje en la pantalla:
Nombre completo
Numero de matricula
Fecha actual del sistema

## Elemento de decision propia
Se realizo un programa adicional que convierte unidades de eleccion propia.

## Justificacion
Se eligio este programa porque permite practicar operaciones y uso de variables y eso hace que me ayude a saber un poco de todo sobre java.
